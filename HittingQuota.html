The Gamma Distribution¶

Math variables¶

I like to think of the gamma parameterized with $\alpha$ and $\beta$, but scipy uses $\alpha$ and $\theta$.

  • alpha: events until completion.
  • beta: event rate.
In [2]:
alpha = 5
beta = 0.5
theta = 1 / beta

Business variables¶

In [3]:
remaining_quota = alpha
success_rate = beta
remaining_opportunities = 5

grv = stats.gamma(remaining_quota, scale=theta)

Interpretation¶

The cumulative distribution function gives us the probabiility of filling the remaining quota in $x$ or fewer opportunities.

In [4]:
grv.cdf(remaining_opportunities)
Out[4]:
np.float64(0.10882198108584877)

Conversely, the survival function gives us the probabiility that the quota takes more than $x$ opportunities.

In [5]:

We can use this technique for the entire business.¶

# Let's set quota to 90% of 28 day expected total by default.
def set_quota(daily_sales_rate, workdays=28, quota_threshold=0.9):
    quota = (
        daily_sales_rate * workdays * quota_threshold
    ) // 1  # Flooring for an integer.
    return quota
In [7]:
def create_quota_frame(daily_sales_rate, workdays):
    quota = set_quota(daily_sales_rate, workdays)
    daily_sales = stats.poisson(daily_sales_rate).rvs(workdays)
    
    df = pd.DataFrame(
        {
            "daily_sales_rate": daily_sales_rate,
            "quota": quota,
            "daily_sales": daily_sales,
        }
    ).assign(
        days_remaining=lambda df: (workdays - 1) - df.index,
        mtd_sales=lambda df: df.daily_sales.cumsum(),
        quota_remaining=lambda df: (df.quota - df.mtd_sales).clip(lower=0),
        quota_reach_probability=lambda df: stats.gamma(
            df.quota_remaining, scale=(1 / df.daily_sales_rate)
        ).cdf(df.days_remaining),
        quota_reached=lambda df: df.quota_remaining == 0,
    )
    return df

def plot_quota_probability(selected_row):
    sample_day = selected_row
    
    grv = stats.gamma(
        sample_day.quota_remaining,
        scale=(1/sample_day.daily_sales_rate)
    )
        
    grv.cdf(sample_day.days_remaining)
    
    x = np.linspace(grv.a, 20, 1000)
    y = grv.pdf(x)
    
    ax = sns.lineplot(x=x, y=y)
    ax.fill_between(x, y, where=(x <= sample_day.days_remaining), alpha=0.5)
    title = f'Daily sales rate: {sample_day.daily_sales_rate:.3f}; Days remaining: {sample_day.days_remaining}\nProbability of filling {sample_day.quota_remaining:.0f} remaining sales: {grv.cdf(sample_day.days_remaining):.3f}'
    ax.set_title(title)
